Editores HTML gratuitos

Un gran número de editores HTML están disponibles para su descarga en Internet. Los programas gratuitos no son inferiores a las versiones de pago en cuanto a sus funciones básicas.

  • Notepad ++: editor HTML slim y simple, tiene una función de resaltado de sintaxis y autocompletar. Además, se pueden editar varios documentos en varias fichas.
  • phase 5: un editor HTML claramente ordenado, que proporciona numerosas características muy útiles y comodas, como el resaltado de sintaxis y un dispositivo corrector. Además, el código también se puede cargar en una vista de navegador. Este editor es gratuito para uso doméstico, escuelas y oficinas. Sólo está disponible para Windows.
  • NVU / KompoZer: La NVU es un editor WYSIWYG, que sobretodo es adecuado para crear webs. Un cargador FTP integrado te permite subir la página HTML creada directamente a Internet. Un descendiente directo de la misma función es KompoZer. Ambos programas pueden utilizarse con Windows, Linux e iOS.
  • Google Web Designer: Este software gratuito de Google ofrece funciones completas y también es adecuado para el diseño de sitios web completos. Se puede utilizar tanto en modo WYSIWYG como en vista de código para HTML, CSS, JavaScript o XML. Además, el Diseñador web de Google admite la inserción de objetos 3D y HTML 5. Es compatible con Windows e iOS.
  • Amaya: El editor HTML gratuito del W3C es particularmente notable porque incluye estándares que aún están en desarrollo. Por esta razón, Amaya es adecuado para demostraciones de softwares nuevos.
  • Netbeans: El editor HTML gratuito Netbeans trabaja dentro de un IDE (entorno de desarrollo integrado). Está disponible para Windows, iOS y Linux y también es compatible con HTML5. La corrección automática de errores ayuda a evitarlos.
  • Hex Editor: Este editor HTML está destinado a los profesionales que desean editar el código de control en el sistema hexadecimal además del código HTML.
  • Microsoft Expression Web 4: Este extenso software de Microsoft permite la creación de páginas web completas. Puedes seleccionar la vista de la fuente o un editor WYSIWYG. Una función de corrección inteligente para HTML y PHP está integrada.
